# magic_4K
Python实现的数学魔术，演示4张K牌的最终状态必然性。

---

## 功能介绍
- 演示 **正反交错重组**、**S形铺放**、**方向翻折** 等数学变换步骤。
- 最终展示所有操作后的结果。

### **执行代码**
```bash
python magic_prove2.py
```
